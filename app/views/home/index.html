Panda Links

Simple, fast link organisation

About

Panda is an application for organising web links. The application is developed in order to learn about state of the art javascript development. The application uses javascript for implementing both, backend and frontend.

Features

  • Create, update, delete web links
  • Search web links
  • Simple user management
